# Mazda MX-5 Miata NC

The NC is the third generation of the Mazda MX-5 Miata.
It arrived for the 2005 model year and is easy to recognize in this test set
because it introduced the Power Retractable Hard Top, also called PRHT.

Key facts for this test document:
- Generation code: NC
- Production era: 2005 to 2015
- Unique roof option: Power Retractable Hard Top (PRHT)
- Common engine mention in enthusiast summaries: 2.0-liter inline-four
- Test hint: only the NC document in this folder talks about PRHT

Simple summary:
If a question mentions PRHT or asks which Miata generation brought a retractable
hard top option, the source should be this NC document.
